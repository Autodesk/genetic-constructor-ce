Module: api

window.constructor.api

Constructor provides a client API for third parties to interact with its data.

This object merges all the Actions and Selectors into a single object, keyed by type of data (e.g. blocks, projects, ui).

See Actions and Selectors for functions available on this object.

Internally, all actions and selectors are automatically bound so they automatically dispatch, without requiring the Redux wrapper syntax store.dispatch(<action>). See Store for more information on Redux.

//create a project with a single block in it, and rename the block

const block = constructor.api.blocks.blockCreate({metadata: {name: 'First Name'}});

const project = constructor.api.projects.projectCreate({components: [block.id]});
